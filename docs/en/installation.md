# Installation Guide

## 1. Clone Repository

```bash
git clone https://github.com/Zilong-L/EarTrainer.git
```

## 2. Install Node.js

Install Node.js v22 using Node Version Manager (NVM):

```bash
nvm install 22
nvm use 22
```

## 3. Install Dependencies

```bash
npm install
```

## 4. Start Development Server

```bash
npm run dev
```

This will run the application locally.
